PLAN POUR LE PROJET PIPEX :

Faire en sorte de decouper les commandes, surement grace a split.
Il faut que les commandes soit decouper chaques espaces, ex : ls -ls 
arg = "ls" "-l"
pour pouvoir les utiliser avec execve.

Pour pouvoir donner un fichier en entree a une commande, il suffit de mettre le fichier apres la commande ex : 
pour grep : arg = "grep", "text", "test.txt" 

Donc pour utiliser execve il faut trouver le chemin du binaire de la commande, puis avoir tout les arguments,
en mettant derriere le fichier a partir duquel lire. 
trouver le chemin : witch "la commande"

Puis se renseigner sur comment faire le pipe.

Puis faire la commande suivante avec en entree ce que le pipe renvoie. puis mettre ce que la deuxieme commande renvoie
dans le fichier outfile.
